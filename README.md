# 🐘 PostgreSQL Learning Repository

A structured collection of PostgreSQL queries and examples covering core database concepts, from basic table operations to advanced topics like joins, transactions, and role management.

This repository is designed as a hands-on reference for learning and revising SQL using PostgreSQL.

---

### 🏗️ Data Definition (DDL)
- Creating and modifying tables
- Altering schemas
- Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL)

### ✏️ Data Manipulation (DML)
- Insert, Update, Delete operations
- Upserting data
- Inserting multiple rows

### 🔍 Querying & Filtering
- WHERE conditions
- Logical operators
- Sorting and limiting results
- GROUP BY and HAVING
- Aggregate functions
- Subqueries
- BETWEEN, IN, LIKE, and IS NULL

### 🔗 Joins & Relationships
- INNER JOIN
- LEFT JOIN
- FULL OUTER JOIN
- SELF JOIN
- CROSS JOIN
- Working with schemas

### 🧠 Advanced SQL Functions
- `CASE`
- `CAST`
- `COALESCE`
- `NULLIF`
- Window functions

### 🔁 Control Flow
- Conditional logic inside queries
- Loops and control statements
- IF statements

### 🔒 Transactions & Constraints
- BEGIN / COMMIT / ROLLBACK
- Ensuring data integrity
- CHECK constraints
- UNIQUE constraints
- NOT NULL constraints
- FOREIGN KEY constraints
- PRIMARY KEY constraints

### 🗂️ Schemas & Organization
- Managing schemas
- Structuring databases
- Altering schemas

### 👤 Roles & Permissions
- Creating roles
- Granting and revoking access
- Role memberships
- Managing role privileges

### 🆕 Additional Topics

### 🛠️ User-Defined Functions
- Creating functions
- Dropping functions
- Function overloading
- Function parameters
- Returning tables

### 🔢 Working with Sets
- UNION, INTERSECT, and EXCEPT
- Set operations with duplicates

### 🔍 Subqueries and CTEs
- Writing subqueries in SELECT, FROM, and WHERE clauses

### 📦 In-Built Functions
- String functions (e.g., `LENGTH`, `CONCAT`)

## ▶️ How to Use

1. Install PostgreSQL locally.
2. Open `psql` or any PostgreSQL client.
3. Run queries from the files step by step.
4. Alternatively, download the PostgreSQL extension for VS Code to work directly within the editor.
