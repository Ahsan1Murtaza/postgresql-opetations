# 🐘 PostgreSQL Learning Repository

A structured collection of PostgreSQL queries and examples covering core database concepts, from basic table operations to advanced topics like joins, transactions, and role management.

This repository is designed as a hands-on reference for learning and revising SQL using PostgreSQL.

---

## 📚 Topics Covered

### 🏗️ Data Definition (DDL)
- Creating and modifying tables
- Altering schemas
- Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL)

### ✏️ Data Manipulation (DML)
- Insert, Update, Delete operations
- Filtering and querying data

### 🔍 Querying & Filtering
- WHERE conditions
- Logical operators
- Sorting and limiting results

### 🔗 Joins & Relationships
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Working with multiple tables

### 🧠 Advanced SQL Functions
- `CASE`
- `CAST`
- `COALESCE`
- `NULLIF`

### 🔁 Control Flow
- Conditional logic inside queries

### 🔒 Transactions & Constraints
- BEGIN / COMMIT / ROLLBACK
- Ensuring data integrity

### 🗂️ Schemas & Organization
- Managing schemas
- Structuring databases

### 👤 Roles & Permissions
- Creating roles
- Granting and revoking access

---

## 🛠️ Tech Stack

- PostgreSQL

## ▶️ How to Use

1. Install PostgreSQL locally
2. Open `psql` or any PostgreSQL client
3. Run queries from the files step by step
